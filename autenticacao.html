Autenticação

Atualmente é possível autenticar na API do Padmoney usando Token de Acesso:

Token de Acesso

Para poder usar a autenticação via Token de Acesso, você precisa gerar e pegar o seu Token, onde será necessário informar uma Chave de Segurança, nas configurações da sua conta.

Ainda com dúvidas para gerar o token? Acesse nossa Central de Ajuda!

Após a geração do Token, com a Chave de Segurança e o Token em mãos, basta enviar no header da requisição essa informações.

Exemplo de requisição autenticada (onde QwertYuiopAsdfghjklZxcvBnm é o Token de Acesso da conta e Br3z0 é a Chave de Segurança): 

GET /v2/ping
Padmoney-Token: QwertYuiopAsdfghjklZxcvBnm
Padmoney-Token-Secret: Br3z0
