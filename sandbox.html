Ambiente de Testes

O Padmoney oferece um ambiente de testes (Sandbox) para uso da API.

Este ambiente é completamente separado do ambiente de Produção, e nenhum dado é compartilhado entre os ambientes.

Para começar a utilizar o ambiente Sandbox, é necessário criar uma conta através do endereço https://dev.padmoney.com/system/conta/cadastrar.

Após criar a conta, você poderá utilizar o ambiente Sandbox da mesma forma que usará o ambiente de Produção.

Nenhuma validação de conta é realizada neste ambiente e logo após a criação da conta, você já conseguirá utilizar a API.

Atenção que o token de acesso da conta no ambiente Sandbox é diferente do token da sua conta no ambiente de Produção.

Endpoint do Sandbox

A URL Base do ambiente Sandbox é https://dev.padmoney.com/api/v2/

Todas as requisições devem usar o schema https por questões de segurança.
